Python Version: 3.8
PyCharm Professional

1) Extract the modules to the same folder as Reuters dataset.
2) Run the inverted_index module.
3) Run the query_processor module.
4) To make a proximity search, enter two words with a number between them (w1 k w2). Words and the number must be separated by whitespace. 
5) To make a phrase search, enter phrase between double quotation marks (""). Words must be separated by whitespace.
6) If a word that is not included in the dictionary is searched, no results will be returned.
